# 指纹浏览器工具生态指南

面向指纹浏览器用户的中文工具生态研究与项目索引，持续整理指纹浏览器、代理IP、浏览器与网络环境检测、SMS接码与验证、临时邮箱、虚拟信用卡、随机测试数据、云手机及浏览器自动化工具。

本账号采用“一项主题，一个仓库”的方式组织内容。各主题仓库分别记录评估方法、产品价格、版本变化、来源日期与可核查证据，避免把不同工具简单堆积成一份推荐清单。

## 当前项目

### [2026指纹浏览器评测与选择指南](https://github.com/ZkingWorkgo/fingerprint-browser)

整理主流指纹浏览器与反检测浏览器的功能、价格、浏览器内核、平台支持、自动化能力、团队功能和版本变化。

- **当前状态：** 评估框架与首轮公开资料核查阶段
- **覆盖方向：** 指纹环境、隐私与数据安全、自动化、团队协作、平台兼容性及价格
- **项目内容：** 评估方法、产品资料、来源证据、变化记录与后续测试结果

## 信息如何整理

项目优先使用产品官网、帮助中心、价格页面、开发文档和公开政策作为来源，并记录核查日期。不同类型的信息使用以下方式区分：

| 信息标签 | 含义 |
|---|---|
| 官方资料 | 厂商官网、帮助中心、价格页或开发文档中的公开说明 |
| 公开核查 | 对多个公开来源进行交叉检查后确认的信息 |
| 编辑分析 | 基于现有证据作出的解释、比较或适用场景判断 |
| 实际测试 | 已公开测试环境、步骤、日期、截图或日志的测试结果 |
| 待核实 | 暂时缺少足够证据，不作为确定结论使用的信息 |

价格、功能、国家覆盖和平台规则可能随时发生变化。使用具体信息前，请查看对应仓库中的来源日期和最近更新时间。

## 项目原则

- 不将厂商宣传直接写成编辑结论
- 不虚构测试过程、成功率、用户反馈或原始数据
- 不在缺少证据时使用“最佳”“最安全”或“第一”等绝对结论
- 对可能变化的价格、政策和功能标注核查日期
- 发现信息错误或版本变化时保留必要的修订记录

## 使用边界

这些项目服务于合法的隐私保护、软件测试、跨境业务、广告验证、环境研究与自动化工作流。内容不提供批量注册、账号交易、真实身份数据、KYC规避或其他违反法律及第三方平台规则的操作指导。

## 更新与纠错

如发现价格、功能、版本或来源信息已经变化，请在对应主题仓库提交Issue，并尽量附上官方页面、公开文档、截图日期或可以验证的复现步骤。
